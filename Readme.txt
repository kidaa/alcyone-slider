Easy to use Image slider.

== Description ==
Making custom slider with Alcyone Slider never been easier. When creating new slider you can choose custom width and height or  some of predefined custom sizes. After choosing some of custom styles like duration between picture, transition type (fade, horizontal, vertical...), autoplay, etc. you can preview it before publishing it on site.
You can also choose between available slides (images you cropped to fit your slider) and active slides (images currently on your active slider).

Some of functions:
*  All browsers support
*  Slide transition
*  Simple design
*  No coding needed


------   General settings   -------
*  Transition time
*  Duration
*  Transition type
*  Pause on mouseover
*  Autoplay
*  Stop on action

------- Navigation -------
*  Navigation dots
*  Navigation arrows
*  Mouseover show arrows
*  Mouseover show dots

== Installation ==
1. Upload \"alcyoneslider\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Click on the new menu item \"Alcyone Slider\" and create your first easy slider!
4. Now you can add Alcyone Slider everywhere you want, through widgets or by shortcode.
4. Your custom slider will now appear on the page / post / template , depends where you putt it!
